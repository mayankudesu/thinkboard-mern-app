## ğŸ“– About the Project

**ThinkBoard** is a full-stack note-taking web application that allows users to write, manage, and organize their notes efficiently in one place.

Users can create new notes, edit existing ones, and delete notes whenever required. The application is built using the **MERN Stack**, ensuring smooth client-server communication through RESTful APIs while performing efficient CRUD operations.

It features a clean and responsive user interface designed to provide a seamless user experience across devices.  
The application is deployed on Render for easy online accessibility.
